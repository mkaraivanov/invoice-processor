## Key Decisions

- **Route groups** `(auth)` and `(protected)` apply different layouts without URL segments
- **`(protected)/layout.tsx`** is a Server Component that checks session and redirects unauthenticated users
- **`middleware.ts`** must be at project root (not inside `src/`) вЂ” Next.js requirement
- **`src/app/generated/prisma/`** is the Prisma client output path вЂ” do not edit manually
- **`src/lib/utils.ts`** is generated by shadcn вЂ” do not create manually
